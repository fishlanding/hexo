先讲最难的部分-GhatGPT API的获取

1.API中转站（以便携AI为例）

注册链接：https://api.bianxie.ai/register?aff=4wNQ

国内有许多api中转站，需要注意的是不同中转站对美元的充值比例可能会有不同,下面是便携AI的充值比例：

5RMB=1美元，相当于官网6.8折

最重要的是 模型价格 ，充值之前一点要仔细对比一下中转站和官方的模型价格，以免被坑害。
例如：

gpt-4o-mini 倍率 模型价格
便携AI 模型：0.075
补全：4		 提示 $0.15 / 1M tokens
补全 $0.6 / 1M tokens
Openai官方 模型：0.075
补全：3		 提示 $2.50 / 1M tokens

补全 $7.50 / 1M tokens

这么算下来，使用中转站是比直接调用官方的更便宜，还不用科学上网。但是，使用中转站提供的api接口，可能存在隐私泄露问题。

2.官方代理站（以秦始皇为例）

注册链接：https://api.oaipro.com/register?aff=bXW2

✨ 没有技巧 ✨ 只是官代 ✨
本站没有特色，稳定不折腾，踏踏实实用。折腾过就会确信：时间就是金钱，稳定压倒一切。
直接代理官方接口：https://api.openai.com。实打实付费key，为付费不方便的朋友提供便利。
美卡直接绑官方支付，直来直去。不套娃、不套路、不以次充好、不使用各种 chat2api 项目。
模型全、余额不会过期，随时用。在一些紧要的关头不失为一种很好的兜底手段，有备无患。
试用是没有试用的，因Stripe支付手续费原因，利太薄，权当跟大家交个朋友。
费率跟官方完全一致，所以不另外在此列出，大家可以直接阅读官方的费率文档：
1OpenAI: https://openai.com/pricing
2Claude: https://www.anthropic.com/api
为方便直接使用，已搭建聊天应用，输入本站的key即可对话（已内置本站接口地址）：
1基于 lobe-chat：https://chat.oaipro.com
2基于 ChatGPT-Next-Web：https://nextchat.oaipro.com
已建立专用Telegram交流群，请私聊：@oaipro_bot发送票据截图获取专用交流群邀请链接。
应大家要求，目前已增开Claude官代，Build Plan (Tier 4) 级别。各位可以体验了！
无需注册，直接使用第三方登录登入即可。若登录遇错误：该 LINUX DO 账户已被绑定，可开无痕模式登录即可。
最后说一句，那啥：请自觉遵守《生成式人工智能服务管理暂行办法》 。

这个我就不多说了，直接看下面的API价格对比吧。

便携AI聚合API有三个接入地址，一般推荐选择第一个或者第二个：
●中转API调用地址①（中国香港服务器，直连线路，带宽大）：https://api.bianxie.ai
●中转API调用地址②（国内上海服务器，带宽稍小）：https://api.bianxieai.com
●中转API调用地址③（国外服务器，也可以直连，备用）：https://api.a8.hk
